1.React-router-dom.
==================================================================================================
Позволяет переходить на другие страницы без перезагрузки страницы.

По своей сути, то, что делает React Router, - это условный рендеринг определенных компонентов
для отображения в зависимости от маршрута, используемого в URL-адресе
(/ - для домашней страницы, /about - для страницы "О нас" и т. д.).

npm install react-router-dom - установка библиотеки.

1)<BrowserRouter><BrowserRouter/> - в него необходимо обернуть то, что мы будем использовать для рендеринга.

2)<Routes><Routes/> - Он гарантируют, что одновременно будет отображаться только один компонент.
Если мы не используем его, мы можем по умолчанию использовать компонент Error (страница не найдена).

3)<Route><Route/> - Это ссылки между компонентами, которые должны быть помещены внутри тегов <Switch>.

Атрибуты:

1.path - URL путь к файлу.
2.element - имя элемента, который необходимо загрузить.

Базовая структура для отображения какой-то конкретной страницы:

 <Routes>
        <Route path='/' element={<Home/>}></Route>
 </Routes>
