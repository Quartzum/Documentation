C 2009 года вышел ECMAScript 5 и добавил новые возможности для JS. Начиная с него, чтобы старый код работал
корректно используется строка - "use strict". Ее ставят в старых проектах, где еще не было ECMAScript 5, чтобы
использовать новые возможности.
Или не ставят, если есть вероятность того, что появятся ошибки.

"use strict" - должна обязательно стоять на 1 строчке файла, иначе режим не заработает.

Также нет никакой директивы, которая позволит выйти из этого режима.

"use strict" по умолчанию включена в браузере.

Для того, чтобы обойти "use strict" в браузере, можно использовать след.код в консоли:

'use strict'; <Shift+Enter для перехода на новую строку>
//  ...ваш код...
<Enter для запуска>

Для старых браузеров можно использовать:

(function() {
  'use strict';

  // ...ваш код...
})()
