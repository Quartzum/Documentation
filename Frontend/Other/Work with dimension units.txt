Единицы измерения бывают:
Абсолютные - не зависят от значений других свойств.
Относительные - зависят от значений других свойств.

1.Пиксели.
==================================================================================================
Абсолютная и самая базовая единица измерения.
Бразуеры пересчитывают, при верстке, другие единицы измерения именно в пиксели.
В качестве значений можно использовать дробные числа.

Размера шрифта браузера по умолчанию - 16px.
Свойство font-size наследуется потомками.

Приемущества:
Четкость, понятность, простота для начинающих верстальщиков

Минусы:
По сравнению с относительными единицами измерения создает проблемы при создании адаптивной верстке.
При увеличении размера шрифта пользователем в браузере появляются проблемы(основной шрифт - 16px меняется,
а вот то, что строго задона в CSS свойствах меняться не будет и шрифт в итоге не поменяется).

2.em.
==================================================================================================
Относительная единица, формируется относительного текущего или унаследованного размера шрифта.
1em равен значению font-size.
Его значения можно использовать в дробных числах.

1em по умолчанию = 16px.

Приемущества:

При изменении размера шрифта пользователем в настройках браузера, размер шрифта будет изменятся
адаптивно. Отлично подходит для указания размеров, зависящих от шрифта.
Используется в медиазапросах.

Недостатки:
Требуются доп.вычисления при переносе размеров из макета в пикселях.

3. rem.
==================================================================================================
Относительная единица. Размер формируется относительного размера шрифта в теге <html> или браузера
по умолчанию.
1rem = значению font-size в теге html или браузера по умолчанию.
Можно указывать дробные числа.

Приемущества:

Те же, что и у em, но здесь нет зависимости от наследования.

Недостатки:
Требуются доп.вычисления при переносе размеров из макета в пикселях.

4.%.
==================================================================================================
Относительная единица. Можно указывать дробные числа.
CSS свойства вычисляют % относительно разных значений:

1.Width - относительно ширины родительского элемента.
2.Height - относительно высоты родительского элемента, но только если у него задана высота,
либо это flex или grid.
3.При относительно позицировании(position:relative;+top left bottom right) нужны те же условия.
4.Для абсолютного позиционирования нужны те же условия
5.Padding/margin - относительно ширины родительского элемента, даже если задаются вертикальные отступы.
6.Transform:translate(x,y) - относительно ширины/высоты самого объекта к которому применено свойство.
7.Font-size и line-height - относительно размера свойства font-size.

5.VW,VH,VMIN,VMAX.
==================================================================================================
Работают относительно окна(viewport)браузера.
Viewport, простыми словами, это вот эта рамка через которую мы просматриваем сайт.

Итак VW работает относительно ширины viewport, VH относительно высоты viewport.
VMIN относительно наименьшего значения из(VW,VH).
VMAX относительно наибольшего значения из(VW,VH).

100vw равно ширине окна браузера.
100vh равно высоте окна браузера.
100vmin равно наименьшему из(vw,vh).
100vmax равно наибольшему из(vw,vh).

Приемущества:
Относительность позволяет на порядок уменьшить кол-во действий при адап.верстке.
Не ограничивает пользователя в настройках браузера. МОжно привязываться к размерам окна, что позволяет
верстать отзывчивые интерфейсы.

Недостатки:
Сложность в использовании. Нужно обладать хорошими знаниями адаптивной верстки и возможностей CSS.
На мобильных устройствах, при работе на всю высоту(100vh)требутся дополнительные действия на JS.

6.Прочие единицы измерения.
==================================================================================================
fr - единица измерения в модуле Grid. Указывает какую часть(фрагмент) должен занимать тот или иной
грид-элемент. Значения могут быть дробными. Отлично работает с отступами gap.
ex - единица измерения работающая относительно размера латинского символа "x". Используется редко.
ch - единица измерения работающая относительно размера символа "0". Используется редко.
Единицы MM,CM,PT,PC являются производными от пикселя и уже не используются.

Формулы конвертации:

из px в em - значение в px поделить на значение font-size(текущее или унаследованное).
из px в rem - значение в px поделить на значение font-size указанное для тега HTML(если не указанно, то на 16px).
из px в % - значение в px поделить на значение относительно которого высчитывается процент и умножить на 100.
