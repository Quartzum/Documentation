node -v - посмотреть версию node.js.
node - попасть внутри консоли в интерактивную среду node.js.

Чтобы запустить программу, необходимо в командной строке ввести:
node название(можно без расширения).

npm -v - версия пакетного менеджера npm.
npm init - инициализация прокета.
npm i chalk - установка пакета chalk.
npm i nodemon - установка пакета nodemon.
npm i nodemon -D - установка пакета для разработки.(данные пакеты будут использоваться только на этапе разработки,
а после завершения удаляться из зависимостей).

Пакеты Node.js:
==================================================================================================
1)chalk - форматирование сообщений, выводимых в терминал.
2)nodemon - перезапускает сервер, после изменения файлов приложений.
3)express - фреймворк web-приложений для Node.js.
4)cross-env - в Windows переменные окружения задаются командой SET, в Linux используется команда export,
а в Mac OS команда env. Эту проблему поможет решить пакет cross-env.
5)dotenv - загрузка переменных окружения из файла ".env" в process.env.
6)progress - «терминальный» индикатор прогресса (ASCII-символы).
7)minimist - разбор аргументов, передаваемых через терминал.
8)fs-extra - дополнительные методы по работе с файловой системой (расширение fs).
9)uuid - генерация уникального идентификатора.
10)moment - форматирование даты и времени.
11)axios - отправка HTTP-запросов. Работает как в Node.js, так и в браузере.
12)async - утилита для работы с асинхронным кодом. Работает как в Node.js, так и в браузере.
13)validator - библиотека для валидации и обезвреживания данных, передаваемых в виде строк.
