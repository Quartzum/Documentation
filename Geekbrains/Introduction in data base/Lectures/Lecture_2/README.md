# Знакомство с базами данных
---
---

## SQL - это особый язк программирования, который позволяет формулировать то, что нужно сделать с данными в таблицах.

Используя псевдокод можно представить некоторые операции с базами данных:

`SELECT * FROM Общий список`

`SELECT` - выбери.

`*` - после SELECT указывается набор столбцов, * означает что мы хотим видеть все столбцы.

`FROM` - "из"указываем, откуда необходимо выбрать информацию.

`Общий список`- название таблицы из нашего примера.

`WHERE` - условие.

`ON` - условие.

## Извлечение данных из таблиц:

1. `INNER JOIN` - создание новой таблицы, в которой будет результат соединения всех данных двух других таблиц.
1. `LEFT JOIN` - создание новой таблицы, в которой будут все данные из левой таблицы, и то что ей соответствует из правой.
1. `RIGHT JOIN` - создание новой таблицы, в которой будут все данные из правой таблицы, и то что ей соответствует из левой.
1. `FULL JOIN` - создание новой таблицы, в которой будут данные сопоставимые между правой и левой таблицой.
